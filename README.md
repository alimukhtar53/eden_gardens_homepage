## Usage

Simply open any HTML file in your browser, or use the color customizer to experiment with different color schemes.

## Customization

Use the `color-customizer.html` tool to:
1. Adjust primary and background colors in real-time
2. Choose from preset color schemes
3. Copy your custom color values
